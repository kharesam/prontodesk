# ProntoDesk — Digital Products Business Execution Plan

---

## Phase 1 — Foundation (Week 1)

1. Create a dedicated business Gmail account
2. Choose and verify shop name availability on Etsy, Gmail, and Pinterest
3. Open Etsy shop — location India, currency USD, digital products only
4. Upload basic banner and logo (Canva, 30 minutes max)
5. Add shop policies (use Etsy's default digital download policy template)
6. Install Canva Pro — needed for template sharing links
7. Install research tools — EverBee Chrome extension, eRank free account, InsightFactory free trial
8. Skip Lemon Squeezy for now — revisit in Month 3

---

## Phase 2 — Niche Research (Week 1–2)

1. Run Spy on Rising Stores workflow across 2–3 broad categories
2. Cross-validate signals using eRank and InsightFactory
3. Produce Validated Niche Brief for 2 confirmed niches
4. Do not start building until at least 1 brief is confirmed
5. Start with niches that share a coherent buyer identity (solopreneurs, small business owners) — keeps the single shop strategy clean

---

## Phase 3 — First Product Build (Week 2–3)

1. Build first Canva template product in confirmed niche
2. Aim for a bundle or multi-page product — higher perceived value
3. Export PDF instructions file — US buyers expect clear setup guidance
4. Package files: Canva template link + PDF instructions + any bonus assets
5. Create 8–10 mockup images per listing (Canva or Placeit)
6. First image = lifestyle mockup showing product in use
7. Subsequent images: feature callouts, what's included, compatibility notes

---

## Phase 4 — First Listings on Etsy (Week 3)

1. Write full listings — title, description, all 13 tags (ask Claude for each listing)
2. Price first products at $7–$15 for fast review velocity
3. Publish minimum 3–5 listings before considering the shop live
4. More listings = more search entry points = better early ranking

---

## Phase 5 — Early Traction (Week 4–8)

1. Start Pinterest business account — pin every product, 3–5 pins per listing
2. Ask Claude for pin descriptions optimised for niche keywords
3. Drive first 10 sales — use a launch discount (30–40% off) to accelerate
4. Get first 5–10 reviews — critical for Etsy search ranking
5. Monitor weekly in EverBee — views low means SEO issue, views high but no sales means images or price issue
6. Report numbers to Claude for diagnosis and iteration

---

## Phase 6 — Add Creative Market (Month 2)

1. Adapt existing Etsy listings for Creative Market buyer (professional designers — more technical language, emphasise file quality and editability)
2. Upload same products — adaptation takes ~1 hour per product
3. Accept the 50% commission — buyer quality justifies it
4. Do not create new products specifically for Creative Market at this stage

---

## Phase 7 — Own Traffic Channel (Month 3)

1. Set up Lemon Squeezy account — connect Indian bank account
2. Mirror your best-selling Etsy products on Lemon Squeezy
3. Start driving Pinterest traffic to Lemon Squeezy checkout links (bypasses Etsy fees)
4. Begin WooCommerce site build — basic store, SEO-optimised product pages
5. Add Gumroad or Payhip as backup checkout if needed

---

## Phase 8 — Scale (Month 4–6)

1. Add product variations within winning niches — build listing volume
2. Target 20–30 active listings across 2–3 niches
3. Expand to a third niche only after sales data validates the first two
4. Consider a second Etsy shop only if a niche feels completely off-brand for the main shop
5. No paid advertising until Month 6 — organic must validate the product first
6. Revisit Design Bundles or Craft Circle only if SVG bundles become a significant catalog share

---

## Platform Sequencing

| Month | Platform | Action |
|---|---|---|
| 1 | Etsy only | Full focus — first 10 sales, first reviews |
| 2 | + Creative Market | Adapt and upload existing listings |
| 3 | + Lemon Squeezy + WooCommerce | Own traffic channel opens |
| 4+ | Scale | More listings, third niche, consider Gumroad/Payhip |

| Timeframe | Target | How |

| Month 1–2 | First sales, first reviews | Etsy organic, launch discount |
